# JD Project

Azure AKS Multi-Region Terraform Infrastructure with Hub-Spoke, Security, Monitoring, and CI/CD.

## Components
- 3 AKS Clusters
- Hub-Spoke Networking
- Azure Firewall & App Gateway
- Traffic Manager
- Key Vault
- Monitoring & Alerts
- GitHub Actions CI/CD

## Usage
```
terraform init
terraform plan
terraform apply
```
